## 项目概述

吸血鬼幸存者风格的修仙/rogue-like塔防游戏。纯代码构建（GameBootstrap 模式），无 prefab 和手动场景编辑。3D 渲染 + 2D 正交相机视角。

## 当前功能清单

### 子弹系统
- 5 种子弹类型同时开火：**Straight**(直射)、**Orbital**(环绕)、**Ricochet**(弹射)、**Shotgun**(散射)、**Chain**(链式)
- 按键 **1-5** 独立 toggle 每种子弹（ON/OFF）
- 顶部 UI 显示：`1:Straight[ON] 2:Orbital[ON] 3:Ricochet[ON] 4:Shotgun[ON] 5:Chain[ON]`
- 所有子弹使用修仙飞剑外观（剑身 Cube + 剑格 Cube + TrailRenderer）
- 命中特效：URP Particles/Unlit 粒子系统

### 波次系统
- 默认每 100 击杀触发一次"突破"升级
- 选增益时 **全局暂停**（GameManager.IsPaused：暂停所有子弹移动、敌人移动、敌人伤害、敌人刷新、子弹发射）
- 每次随机 3 个增益选项，选择后立即锁定不变化
- 选择后进入下一波，敌人属性自动缩放

### 增益系统（10 种）

| 灵气复苏 | 恢复 30% 生命 |

- 底部：FPS + 帮助文字

## 架构要点

- `GameBootstrap.Awake()` 按序创建：Camera → Player → BulletManager → EnemySpawner → GameManager → Canvas(UI) → WaveManager
- 全局暂停通过 `GameManager.IsPaused` 静态字段实现，所有 Update() 入口检查
- 增益系统分两级：全局增益(WaveManager 累积 multiplier → BulletManager.ApplyUpgrades) + 单类型增益(WaveManager → BulletManager.AddXxx 方法)
- WaveConfig 中 `UpgradeDef.targetBulletType` 为 null 表示全局，非 null 表示针对特定子弹类型

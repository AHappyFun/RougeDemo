## 项目概述

吸血鬼幸存者风格的修仙/rogue-like塔防游戏。纯代码构建（GameBootstrap 模式） + Prefab 资源（`Resources/`）。3D 场景 + 俯视角透视相机 + WASD 自由移动。

## 架构要点

- `GameBootstrap.Awake()` 构建顺序：Camera → Scene(可选) → Player(+Movement) → Camera3D(+Follow) → BulletManager → EnemySpawner → GameManager → UI → WaveManager
- `MeshGenerator` 拆分 FromPrefab / Procedural 双路径
- 全局暂停 `GameManager.IsPaused`，所有 Update()/OnTriggerStay 检查
- 增益：全局累积 → `ApplyUpgrades()`；单类型 → `AddXxxBonus()`
- 资源自动生成：ResGenerator + ResSceneGenerator
- 自定义 Shader：Rouge/Character + Rouge/VFX
- Y-up 3D：`Euler(0,angle,0)`，XZ 平面移动
- 场景双模式：SceneBuilder 运行时构建 / ResSceneGenerator 生成 .unity 文件
